=head1 App::SourceCodeStats

Script to show the statistics (such as lines of code, total size in bytes, lines of commants, etc) for a project.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
